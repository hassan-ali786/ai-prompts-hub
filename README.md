# AI Prompts Hub

A simple and clean website that provides useful AI prompts for students, developers, and creators.  
You can easily copy prompts and use them for study, coding, resumes, and content creation.

---

## What this project does

- Provides ready-to-use AI prompts  
- Helps in study, coding, and productivity  
- Includes search feature  
- One-click copy prompts  
- Clean and simple user interface  

---

## Prompt Categories

### Study
- Notes generation  
- Exam preparation help  
- Topic explanations  

### Coding
- Python programs  
- Debugging help  
- Code explanation  

### Career
- Resume improvement  
- Interview preparation  
- Job guidance  

### Content
- LinkedIn posts  
- Social media ideas  
- Content prompts  

---

## Features

- Search prompts  
- Copy with one click  
- Beginner friendly  
- Dark theme UI  
- Fast and simple design  

---

## How to use

1. Open the website  
2. Search any prompt  
3. Click the copy button  
4. Paste and use anywhere
